# Cómo subir a GitHub Pages

1. Descomprimí este ZIP
2. Subí TODOS los archivos a tu repositorio GitHub (reemplazando los existentes)
3. Commit + Push
4. En el navegador: abrí la app y hacé **Ctrl+Shift+R** (o en móvil: Settings > Clear Cache)
   para forzar que el browser descarte la cache del service worker
